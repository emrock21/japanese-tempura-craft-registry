# Japanese Tempura Craft Registry

This registry documents authentic Japanese tempura traditions.  
Tempura is not simply “fried food” — it is a refined craft shaped by regional oils, batter techniques, ingredient selection, and historical lineages.

## Purpose

The goal of this contract is to preserve real tempura knowledge in a permanent and verifiable format.  
Tokyo’s sesame-oil Edo style, Kyoto’s kaiseki-style tempura, Osaka’s kushikatsu lineage, and regional seafood variations each represent distinct culinary identities.

By recording these styles on-chain, the registry protects culinary heritage from distortion and keeps the connection between region, method, and cultural meaning.

## Why this registry matters

Tempura has deep regional differences:

- Edo-style uses sesame oil and ultra-thin batter  
- Kyoto-style is pale, delicate, and tied to kaiseki  
- Osaka kushikatsu uses skewers and panko  
- Nagasaki tempura carries early Portuguese influence  
- Shizuoka sakura-ebi tempura exists only in Suruga Bay  
- Hokkaido seafood tempura highlights cold-water sweetness  

This registry helps by:

- preserving verified information in a neutral and permanent format  
- linking each style to its historical shop or lineage  
- offering a reference for researchers, chefs, and cultural institutions  
- preventing oversimplification of regional culinary identity  

## What makes this registry unique

The registry focuses on the craft process itself:  
batter composition, oil type, frying temperature, ingredient selection, and cultural uniqueness.

The contract is intentionally minimal.  
It records only the essential information needed to identify and contextualize each tempura style.

## How it works

Each entry includes:

- region  
- historical shop or family  
- style name  
- ingredients  
- preparation method  
- what makes the style unique  

A simple like/dislike system allows basic community feedback.
